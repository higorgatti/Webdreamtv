🚀 COMO USAR (PASSO A PASSO):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  INICIAR AMBIENTE LOCAL:

   → Duplo clique em: ABRIR_LOCAL.bat

   Isso vai:
   - Verificar se Apache está rodando
   - Iniciar XAMPP se necessário
   - Abrir http://localhost/webplayer_dev/themes/dreamtv/ no navegador


2️⃣  FAZER MUDANÇAS:

   → Abra index.php no seu editor favorito:
     - VSCode
     - Notepad++
     - Sublime Text
     - Ou qualquer editor

   → Faça as alterações que desejar

   → Salve o arquivo

   → No navegador: CTRL + F5 (força reload)

   → Teste as mudanças

   → Repita até ficar satisfeito


3️⃣  ENVIAR PARA PRODUÇÃO:

   → Quando estiver TUDO funcionando localmente

   → Duplo clique em: UPLOAD_PRODUCAO.bat

   → Confirme o upload digitando "S"

   → Aguarde o upload completar

   → Acesse: http://play.fivetv5.com/themes/dreamtv/

   → Pressione CTRL + F5 para ver as mudanças
